2. Установить Docker.

# docker pull ubuntu

4. * Используя Dockerfile, собрать связку nginx + PHP-FPM в одном контейнере.
$ mkdir my_nginx_fpm
$ cd my_nginx_fpm
cat > Dockerfile
```
FROM debian
MAINTAINER romanl
RUN apt-get update
RUN apt-get install nginx -y
RUN apt-get install php-fpm -y
VOLUME "/var/www/html"
EXPOSE 80
CMD /usr/bin/nginx -g "daemon off;"

```
$ sudo docker build -t my_nginx_fpm .
$ sudo docker run -d -p 82:80 my_nginx_fpm
