## Задание:

1. Управление пользователями:
a) создать нового пользователя;
b) убедиться, что информация о нем появилась в соответствующих файлах в системе;
c) удалить созданного пользователя;

2. Управление группами:
a) создать группу;
b) попрактиковаться в смене групп у пользователей;
c) добавить пользователя в группу, не меняя основной;
d) удалить пользователя из группы.

3. Работа с группами.
a) Создать пользователя с правами суперпользователя. Проверить результат.
b) * Создать группу developer и нескольких пользователей, входящих в неё. Создать директорию для совместной работы. Сделать так, чтобы созданные одними пользователями файлы могли изменять другие пользователи этой группы.
c) * Создать в директории для совместной работы поддиректорию для обмена файлами, но чтобы удалять файлы могли только их создатели.

В качестве решения необходимо прислать текстовый файл (.txt) с набором используемых команд и их результатами (можно скопировать текст из консоли).
Задания со звездочкой дополнительные, необязательные.

Решение:
1.
a) создать нового пользователя;
romanl@gb-vbox:~$ sudo useradd -m -d /home/user1 -s /bin/bash user1
romanl@gb-vbox:~$ sudo passwd user1
Новый пароль:
Повторите ввод нового пароля:
passwd: пароль успешно обновлён

b) убедиться, что информация о нем появилась в соответствующих файлах в системе;
romanl@gb-vbox:~$ cat /etc/passwd | grep user1
user1:x:1001:1001::/home/user1:/bin/bash
romanl@gb-vbox:~$ cat /etc/group | grep user1
user1:x:1001:
romanl@gb-vbox:~$ sudo cat /etc/shadow | grep user1
user1:$y$j9T$f.bQE5uoJk8PHzG4DtKv4.$0PG4S81b0kvKYiBgmXVqsqqQ/1fKBMP00.QfmsA.i6B:19715:0:99999:7:::

c) удалить созданного пользователя;
romanl@gb-vbox:~$ sudo userdel -fr user1
userdel: почтовый ящик user1 (/var/mail/user1) не найден


2.
a) создать группу;
romanl@gb-vbox:~$ sudo addgroup workgroup
Добавляется группа «workgroup» (GID 1002) ...
Готово.

b) попрактиковаться в смене групп у пользователей;
romanl@gb-vbox:~$ sudo usermod -g workgroup user1
romanl@gb-vbox:~$ cat /etc/passwd | grep user1
user1:x:1001:1002::/home/user1:/bin/bash
romanl@gb-vbox:~$ sudo usermod -g user1 user1
romanl@gb-vbox:~$ cat /etc/passwd | grep user1
user1:x:1001:1001::/home/user1:/bin/bash

c) добавить пользователя в группу, не меняя основной;
romanl@gb-vbox:~$ sudo adduser user1 workgroup
Добавляется пользователь «user1» в группу «workgroup» ...
Добавление пользователя user1 в группу workgroup
Готово.
romanl@gb-vbox:~$ cat /etc/group | grep user1
user1:x:1001:
workgroup:x:1002:user1

d) удалить пользователя из группы.
romanl@gb-vbox:~$ sudo deluser user1 workgroup
Удаляется пользователь «user1» из группы «workgroup» ...
Готово.
romanl@gb-vbox:~$ cat /etc/group | grep user1
user1:x:1001:
